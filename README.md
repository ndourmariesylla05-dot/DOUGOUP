# 🌾 DOUGOUB - Plateforme AgriTech Moderne

> **Digitalisation de la filière du mil** : Connecter producteurs, transformateurs et consommateurs à travers une plateforme intuitive et performante.

---

## 🚀 Genèse et Implémentation du Projet

Voici le récit de la création de cette application, de l'idée initiale à la mise en ligne.

### 1. ✍️ Stratégie de Prompting
Le projet a débuté par la définition d'une vision claire. Un premier prompt a été élaboré pour capturer l'essence de l'application :
- **Concept** : Une plateforme complète avec sidebar, menu riche et navigation fluide.
- **Technologie** : Utilisation de `json-server` pour des données mockées dynamiques.

#### 💡 Le Prompt Optimisé
Pour obtenir le meilleur résultat, un prompt professionnel a été conçu et affiné avec l'aide de **ChatGPT** pour être plus percutant :

```text
Crée une application mobile moderne et entièrement fonctionnelle pour un projet AgriTech nommé DOUGOUP.
L’application doit être réaliste, interactive et prête à être utilisée avec des données mockées.
... (Sidebar complète : Accueil, Tableau de bord, Production, Stocks, Marketplace, etc.)
... (CRUD complet, Interactions réelles, Design Marron #7a4e2d et Beige #d8c3a5)
```

### 2. 🎨 Design-to-Code (Figma & Make)
L'étape suivante a consisté à transformer le design en code :
- Utilisation de **Figma** pour la conception visuelle.
- Utilisation du plugin **Make** pour générer automatiquement une structure de code React prête à l'emploi.
- Téléchargement et extraction du projet (Zip).

### 3. 🤖 Raffinement avec Antigravity AI
Une fois le code généré, l'intelligence artificielle **Antigravity** (IDE Agentic) a été utilisée pour :
- **Débogage** : Correction des erreurs post-génération.
- **Optimisation** : Finalisation des composants et de la navigation.
- **Exécution locale** : Validation de l'application en environnement réel.

### 4. 🌐 Versioning et Déploiement
Le projet a été finalisé par une mise en service professionnelle :
- **Git** : Création d'un dépôt pour le suivi de version.
- **Vercel** : Déploiement automatisé pour rendre l'application accessible en ligne.

## 🛠️ Installation et Lancement

### Prérequis
- Node.js (v18+)
- npm ou pnpm

### Démarrage Rapide
1. **Installer les dépendances** :
   ```bash
   npm install
   ```
2. **Lancer le serveur de développement** :
   ```bash
   npm run dev
   ```
